Installation 
===============================

PyPi
===============================

.. code-block:: bash

    pip install cf-explanations


Manual Installation
==============================

Download or clone the repository:

.. code:: bash
    git clone https://github.com/wangyongjie-ntu/Counterfactual-Explanations-Pytorch
    cd Counterfactual-Explanations-Pytorch
    pip install -e .
